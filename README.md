# Italia Restaurant Platform

Stack:
- Laravel 11 API
- React 18 + Vite + Tailwind CSS
- MySQL 8
- Laravel Sanctum
- Stripe
- Docker Compose

## Lancement

```bash
docker compose up --build
```

Applications:
- Frontend: `http://localhost:5178`
- Backend API: `http://localhost:18000`
- phpMyAdmin: `http://localhost:18080`
- Mailpit: `http://localhost:18025`
